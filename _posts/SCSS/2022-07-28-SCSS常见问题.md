## sass计算warning

```css
@use "sass:math";
// math.div in sass

//banner 资源位置计算
@mixin bannerWSP($img, $width, $height, $bgW: 0, $bgH: 0, $bx: 0, $by: 0) {
    width: getPxHalf($width);
    height: getPxHalf($height);
    background: url($img) no-repeat getPxHalf($bx) getPxHalf($by);
    background-size: getPxHalf($bgW) getPxHalf($bgH);
}
@function getPxHalf($val){
    @return math.div($val, 2) + px;
}
```
