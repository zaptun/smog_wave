### 自动播放视频

```typescript
// create a video texture from a path
const texture = PIXI.Texture.from('examples/assets/video.mp4');

// create a new Sprite using the video texture (yes it's that easy)
const videoSprite = new PIXI.Sprite(texture);

//取出node节点
const videoNode = texture.baseTexture.resource.source;
//关键就是设置静音
videoNode.muted = true;
//关闭循环播放
videoNode.loop = false;

// Stetch the fullscreen
videoSprite.width = app.screen.width;
videoSprite.height = app.screen.height;

app.stage.addChild(videoSprite);
```



### animation实现倒放

关键就是 **timeScale**

```typescript
//加载banner03资源
dbfactory.parseDragonBonesData(res.banner03_bones.data);
dbfactory.parseTextureAtlasData(res.banner03_texture.data, res.banner03_texture_png.texture);

//生成banner03实例
const banner03Display = dbfactory.buildArmatureDisplay("banner03");

//设置参数
banner03Display.scale.set(0.5);
banner03Display.x = app.screen.width / 2;
banner03Display.y = app.screen.height / 2;

//放banner03入舞台
app.stage.addChild(banner03Display);

//设置成倒放
banner03Display.animation.timeScale = -1;
//播放动画
banner03Display.animation.play("card03in");
```
