### 1. 连接数据

创建一个TypeORM的配置

```
{
  "type": "数据库类型",
  "host": "数据库主机",
  "port": "数据库端口",
  "username": "数据库用户名",
  "password": "数据库密码",
  "database": "数据库名称",
  "entities": [
    "src/entities/*.ts"
  ],
  "migrations": [
    "src/migrations/*.ts"
  ],
  "cli": {
    "entitiesDir": "src/entities",
    "migrationsDir": "src/migrations"
  }
}
```
